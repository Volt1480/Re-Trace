## 🚀 Overview

**Re:Trace** is a forensic-based anti-cheat scanner designed to detect suspicious execution artifacts and traces on Windows systems.

Instead of intrusive kernel-level anti-cheat techniques, Re:Trace focuses on:

- Execution traces
- Residual artifacts
- Behavioral correlation

Built for **FiveM communities**, but designed to be modular and extensible.

---

## 🧠 Key Concept

> "Cheats can hide, but traces remain."

Re:Trace analyzes:

- What was executed
- Where it came from
- What traces it left behind

## ✨ Features

### 🔍 Artifact Collection
- Prefetch parsing
- Amcache parsing
- Temp & Recent files
- Browser download traces

### 🧠 Detection Engine
- JSON-based rule system
- Regex / pattern matching
- Hash-based detection
- Correlation engine

### 📦 Evidence System
- Structured findings
- Raw artifact evidence
- ZIP packaging
- SHA256 hashing

### 🌐 Backend API
- Scan upload
- Rule distribution
- Multi-tenant ready

### 🖥️ Dashboard
- Scan overview
- Findings explorer
- Evidence viewer
- Filtering & search

### 🎮 FiveM Focus
- Loader detection
- Injector traces
- Temp payload artifacts
- Correlated detection chains
